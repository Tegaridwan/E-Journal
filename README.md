# E-Journal Unipma

#Aplikasi E-Journal Unipma.
#Adalah aplikasi yang membantu siswa prakerin untuk absen dan menginput jurnal setiap kegiatanya.
#Aplikasi ini menggunakan framework flutter dan firebase sebagai databasenya
