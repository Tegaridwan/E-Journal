<h1 align="center">📘 E-Journal Unipma</h1>

<p align="center">
Aplikasi Android berbasis Flutter untuk melakukan absensi dan pengisian jurnal kegiatan siswa prakerin.
</p>

<hr>

<h2>📱 Tentang Aplikasi</h2>

<p>
E-Journal Unipma adalah aplikasi yang membantu siswa prakerin dalam melakukan:
</p>

<ul>
  <li>Absensi harian</li>
  <li>Input jurnal kegiatan</li>
  <li>Monitoring aktivitas prakerin</li>
  <li>Penyimpanan data secara online</li>
</ul>

<p>
Aplikasi ini dibuat menggunakan framework <b>Flutter</b> dan menggunakan <b>Firebase</b> sebagai database backend.
</p>

<hr>

<h2>✨ Fitur Utama</h2>

<ul>
  <li>Login Authentication</li>
  <li>Absensi Siswa</li>
  <li>Input Jurnal Harian</li>
  <li>Riwayat Kegiatan</li>
  <li>Realtime Database Firebase</li>
  <li>User Friendly Interface</li>
</ul>

<hr>

<h2>🛠️ Teknologi yang Digunakan</h2>

<ul>
  <li>Flutter</li>
  <li>Dart</li>
  <li>Firebase</li>
  <li>Android Studio / VS Code</li>
</ul>

<h2>⚙️ Cara Menjalankan Project</h2>

<ol>
  <li>Clone repository ini</li>

  <li>Install dependency Flutter:
    <pre>flutter pub get</pre>
  </li>

  <li>Hubungkan project dengan Firebase</li>

  <li>Jalankan aplikasi:
    <pre>flutter run</pre>
  </li>
</ol>
